Premiers pas




Votre premier projet


Lorsque vous lancez Virtual Art Expo, le logiciel vous demandera si vous voulez créer une nouvelle salle ou en importer une. Dans le cas présent nous créerons une nouvelle salle.


Une nouvelle boîte de dialogue s'ouvre, il vous suffit juste d'appuyer sur "Suivant"


Entrez le nom de votre projet, choisissez son répertoire puis si vous le souhaitez, entrez une description. Cliquez sur "Terminer".


Votre projet est créé. Vous êtes maintenant libre de dessiner votre salle.
