Choisir un point de départ


Vous pouvez choisir un point de départ dans la salle pour votre rendu 3D. Cliquez sur "Placer le point de départ"

Pour accéder au rendu 3D de votre salle il vous suffit de cliquer sur "Passer en vue 3D"
