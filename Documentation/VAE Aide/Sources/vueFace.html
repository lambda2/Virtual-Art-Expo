L'éditeur "Vue de face"


Une fois votre salle fin prête vous pourrez y placer des images. Il vous suffit de verrouiller votrre salle via le bouton "Verrouiller" puis de choisir le mur sur lequel vous voulez placer vos images, en le sélectionnant, et enfin cliquez sur "Vue de face" dans la barre d'outils.




Importer des images


Il vous faut tout d'abord importer vos images dans la bibliothèque de VAE. Pour cela cliquez sur "Importer", sélectionnez les images qui vous intéressent et faites "Ouvrir".


Maintenant que vos images sont importées dans la bibliothèque, vous n'avez plus qu'à faire une glisser-déposer de vos images sur le mur.


Modifier des images


Vous pouvez déplacer vos images en les faisant simplement glisser. Il vous est aussi possible de modifier leur taille, via le coin inférieur droit de l'image ou en entrant les valeurs désirées dans les champs au-dessus. Sachez simplement que les proportions de l'image seront conservées. Il vous est aussi possible de définir à quelle distance se situe l'image par rapport au sol ou au bord du mur sur lequel vous travaillez. De la même manière que pour la vue de haut, vous pouvez supprimer une image en faisant un clic droit sur l'image et "Supprimer".
