Consignes d'installation des fichiers OGRE necessaires pour l'execution du moteur 3D.


1. Compilez les sources

2. Dans le répertoire de l'exécutable de dessin (par exemple : /VAE/VAE-build-desktop/Dessin), copiez-collez 
le contenu du répertoire courant, c'est à dire les fichiers :
	- plugins.cfg (définit les modules que le moteur doit charger au démarrage)
	- resources.cfg (définit les emplacements des fichiers de mesh, de textures et de matériaux au moteur, pointe ici sur le répertoire media)
	- /Media (contient tous les fichiers de mesh, de textures et de matériaux nécessaires)
	
3. Lancez le moteur à partir de l'interface de dessin.
